# Nest.js project starter kit (TypeORM & Postgres)

**Tech Usages:**
```yml
Runtime: Node.js v20 (LTS)
Framework: Nest.js (v10)
Databases: PostgreSQL (TypeORM), and Redis (native-driver)
Mailer: NodeMailer (tempMail)
```
