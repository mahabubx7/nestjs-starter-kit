# Nest.js project starter kit (Knex.js & Postgres)

**Tech Usages:**
```yml
Runtime: Node.js v20 (LTS)
Framework: Nest.js (v10)
Databases: PostgreSQL (knex.js), and Redis (native-driver)

```
